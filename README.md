# ai-music-producer

**Live:** https://benjamin5607.github.io/ai-music-producer/

0코스트 AI 음악 기획 스튜디오 — GitHub Pages에서 바로 쓰는 실사용 워크벤치입니다. 이 프로젝트는 3인 에이전트 페르소나(프로듀서, 작사가, MR 감독)로 구성되며, 각 에이전트는 무료 AI 서비스(ChatGPT, Claude, Gemini, Suno/Udio)를 활용하여 음악 프로젝트를 진행합니다. 프로듀서는 시장 트렌드 분석, 아티스트 컨셉 매칭, 앨범 기획 및 릴리즈 일정 조율을 담당하며, 작가는 프로듀서의 컨셉에 맞춘 가사 작업을 수행합니다. MR 감독은 장르 설정, 코드 진행 가이드 제공, 음악 생성 AI용 최적의 프롬프트 엔지니어링을 담당합니다. 프로젝트는 3인 에이전트 페르소나(프로듀서, 작사가, MR 감독)로 구성되며, 각 에이전트는 무료 AI 서비스(ChatGPT, Claude, Gemini, Suno/Udio)를 활용하여 음악 프로젝트를 진행합니다.

## 무엇이 구현돼 있나
- 3인 에이전트 페르소나 (프로듀서 / 작사가 / MR 감독) 편집
- 단계별 **복사 가능한 실전 프롬프트** (ChatGPT·Claude·Gemini·Suno/Udio)
- 아티스트 프로필·컨셉·가사·MR·녹음 체크리스트·커버/숏츠·유통 체크 전체 플로우
- `localStorage` 프로젝트 저장 + JSON 내보내기
- 무료 툴 바로가기 (BandLab, vocalremover, RouteNote, Amuse 등)

> Dev Studio가 “단계 목록만” 올렸던 깡통 UI를 대체한 버전입니다. 이 프로젝트는 0코스트 AI 음악 기획팀을 구성하여 음악 프로젝트를 진행하는 것을 목표로 합니다.
